git => version control system [download from https:it-scm.com/] -> Standalone Installer

git commands

git init => initialize git

configuration => user name and email
    email => git config --global user.email "sample@gmail.com"
    name => git config --global user.name "sample"

git remote add origin <remote_url>  eg: https://github.com/mern-trainer/sahil-batch.git
git add . => add all files
git commit -m "message" => commit all files

to view remote url => git config --get remote.origin.url
